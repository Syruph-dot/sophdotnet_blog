# A0 故事前提梳理与预警

> 基于 2026 年 5 月可查证的 BA 官方剧情进度（日服主线至 Ex.十字神名篇 第2章 Part3），
> 对本项目设定的每一个核心支点进行"原作事实 vs 创作偏离"的对照检查，并标注风险等级。
> 检索来源：Blue Archive Fandom Wiki、Namuwiki、萌娘百科、NGA 剧情讨论、PTT、Bilibili 考据。

---

## 总览：原作神系三层架构

BA 最底层世界观是一个三层神系更替叙事：

```
无名之神 (Nameless Gods)
 信奉者 → 无名祭司 (Nameless Priests)
    │
    ├─ 被遗忘之神 (Forgotten Gods) 取代
    │     └─ 学生 = 被遗忘之神的化身（持有"神秘"）
    │           └─ 可被"色彩"反转为"恐怖"
    │
    └─ 十字神名 (Decagrammaton)：古代人造 AI 自称为神（独立势力）
```

### 第一层：无名之神（Nameless Gods）
- **时期**：基沃托斯最古老的神明体系，人类尚未为神命名的时代
- **形态**：对天灾地变（打雷、洪水、山火等）的原始崇拜
- **特征**：没有名字 → 无法被呼唤 → 等于"不存在"
- **结局**：被后来"有名字的神明"（被遗忘之神）取代，离开了世界
- **遗留**：超时代科技遗迹「欧帕兹」（Oparts）；阿特拉哈西斯方舟
- **已知代表**：死神阿努比斯（Anubis）

### 第二层：被遗忘之神（Forgotten Gods）
- **时期**：在无名之神之后出现，拥有名字和人格的神明体系
- **与学生关系**：**学生就是被遗忘之神的化身**，每个学生的"神秘"对应某神话体系的神明
- **状态**：名字在现实世界已被遗忘，但在基沃托斯以"学生"形态复活
- **遗产**：圣所之塔、乌特纳匹什提姆之船

### 第三层：色彩（The Colour）
- **来源**：基沃托斯之外的宇宙恐怖实体（克苏鲁式"异次元色彩"）
- **能力**：将"神秘"反转为"恐怖"（本质反转，不可逆）
- **地位**：数秘术的共同敌人，基沃托斯终极威胁

### 中间势力：无名祭司（Nameless Priests）
- 无名之神的信徒，特征：白袍 + 高帽 + 面具，无个体名字（代号 A、B、C）
- **终极目标**：在所有平行世界、所有时空中铲除"被遗忘之神"（即学生）
- **手段**：利用色彩反转神秘为恐怖；建造阿特拉哈西斯方舟
- **造物**：Divi:SION（不可解的军团）、Chesed 等兵工厂 AI
- **王女**：AL-1S（天童爱丽丝）——制造来执行方舟启动的终极兵器

## 核心支点对照

### 支点 1 ：第 0 周目的定义

**你的前提：**
> 此线为零周目，二周目是衍生出的一种可能（此处解释一周目为二周目的 complete dystopian 平行世界）。
> 此线**没有 Sensei，没有色彩**。
> 去神圣化叙事。现实意义导向：找到指导内心的自己。

**原作事实：**
- BA 存在多条平行世界线，官方已确认（终章普雷纳帕特斯 = 另一世界线的老师）
- 另一世界线（即原作中的"bad future"）：老师重伤 100 天→阿拜多斯覆灭→千年毁灭→伊甸条约失败→老师被色彩同化为"色彩的向导者"
- **原作中没有"第0周目"的概念**，只有"平行世界"（不同选择的不同结果）
- 联邦学生会长可能是唯一的跨世界线存在

**偏离分析：**

| 维度 | 原作 | 本项目 |
|------|------|--------|
| 时间层 | 单一时间线 + 平行分支 | 前提性的"第0周目" + 后续周目 |
| 色彩 | 存在的终极威胁 | 不存在 |
| Sensei | 存在的核心角色 | 不存在 |
| 神圣叙事 | BA 的核心结构（超理性救赎） | 被取消 |

"第 0 周目"与"平行世界"的本质区别：

- **平行世界**：同一套规则的不同结果——基沃托斯存在，学生存在，老师可能存在也可能不存在
- **第 0 周目**：规则本身尚未建立——"一切之前"的状态。没有学生、没有神秘、甚至可能没有基沃托斯

**风险等级：🟡 中等**

第 0 周目作为元叙事框架是可行的，但需要明确：

1. 它是"故事发生的世界前史"还是"叙事层的设定说明"？
2. 它与"没有 Sensei 的世界"是不同的概念——没有 Sensei ≠ 没有学生
3. 如果第 0 周目产生了后续周目的基沃托斯，压抑感可能被削弱（压抑变成"设计"而非"自然"）

### 支点 2 ：联邦学生会长失踪的前提

**原作事实：**
- 失踪原因：官方**尚未揭晓**
- 已知信息：请老师来到基沃托斯、留下什亭之匣、失踪数周
- 与阿罗娜的外貌高度一致，但阿罗娜对她的存在"一无所知"
- NGA 社区推测：可能存在上一周目，会长自己做决策导致了坏结局，所以这周目请老师来接手
- **会长的光环即 BA 的标志**，暗示所有世界线中只有一位会长

**原型注释：**
- 什亭之匣（Shittim Chest）= 圣经约柜
- 名字源自什亭木，以色列人进入应许之地前的最后驻扎地
- 密码引用杰里科之战七号角（约书亚记 6 章）

**本项目前提：**
第 0 周目中"没有 Sensei"，学生会长失踪的前提（或学生会长存在与否）需要自行定义。

**风险等级：🟢 低**
原作未揭晓本身就是创作空间。无需填补原作的坑，只需自洽即可。

### 支点 3 ：爱丽丝

**你的前提：**
> 爱丽丝原本为无名（未写完）

**原作事实：**
- 爱丽丝 = **无名祭司制造的终极兵器**，代号 AL-1S，身份为"无名之神的王女"
- 体内 AI Key（原名 Key = "钥匙"），负责引导她觉醒为"王女"，执行阿特拉哈西斯协议
- 被桃井和绿在千禧年郊外的废墟兵器工厂中发现
- 桃井和绿找贝里塔斯（Veritas）伪造学籍让她入学
- **AL-1S 的存在与老师无关**——她在老师来到基沃托斯之前就已存在于废墟中
- Key 终章为保护爱丽丝耗尽能量，后在十字神名篇被复活并获得独立机身

**原型层：**
| 身份 | 原型 | 对应 |
|------|------|------|
| 爱丽丝（学生人格） | 亚里士多德（Aristotle） | "Aris" 取自前四个字母；理性灵魂；求知 |
| Key（AI 人格） | 赫费斯提翁（Hephaestion） | 亚历山大密友；从使命执行者变为守护者 |
| 王女（完整形态） | 亚历山大大帝（Alexander） | "AL-1S" 取自 Alexander；征服/毁灭 |
| 神名层 | 扎格列欧斯→狄俄尼索斯→法涅斯 | 俄尔普斯教三位一体 |

**关键信息：**
- **无名祭司制造了爱丽丝**——爱丽丝本身是"无名"势力的造物
- 她在原作中"获得名字"的过程：桃井和绿给她取名"爱丽丝"
- 爱丽丝的抉择——"我是勇者，不是王女"——是她角色弧光的核心

**对本项目的意义：**

"爱丽丝原本为无名"这个前提可以指向两个方向：

1. **指向她的出身**（与 BA 一致）：她是无名祭司的造物，"无名"即她的出厂设置
2. **指向第 0 周目的状态**：在第 0 周目中，她还只是一个"未激活的程序/可能性"，没有名字、没有身体、没有被赋予人格

如果采用方向 2，需解决：
- 她在第 0 周目以什么形态存在？（纯粹数据？概念？）
- 她如何获得"成为爱丽丝"的可能性？

**风险等级：🟡 中等**
爱丽丝的原作设定已经非常完整且与"无名"密切相关。偏离时需注意不破坏她的核心弧光（自我认同的抉择）。

### 支点 4 ：无名众神与无名祭司

**你的前提：**
> 无名众神、无名祭司到底指什么，原作中的各种暗示？

**原作事实（整合全网资料后）：**

**无名之神（Nameless Gods）**
- 定义：最早的神，人类对天灾地变的原始自然崇拜
- 他们没有名字 → 无法被呼唤 → 等于"不存在"
- 黑服称其为"前启蒙的崇高"
- 科技水平极度发达，拥有"可能性操纵"能力
- 已知代表：阿努比斯（Anubis，也是白子的神秘原型）

**无名祭司（Nameless Priests）**
- 特征：白袍 + 高帽 + 面具，无个体姓名（仅以代号区分）
- 目标：在所有时空中消灭被遗忘之神（学生）
- 手段：色彩、阿特拉哈西斯方舟、Divi:SION 军团
- 憎恨被遗忘之神的原因：无名之神被取代后，他们失去了崇拜对象和"崇高"
- 核心台词（终章）：*"像没有名字、无法被称呼、所以不存在的『无名神明』那样，你们也将达到相同的结局。"*

**被遗忘之神与学生**
- 学生 = 被遗忘之神的化身，每个学生的"神秘"对应一个神明
- 示例：白子 = 阿努比斯 → 反转后 = 黑子（"恐怖"形态）
- 同一个神明的神秘不能同时存在于两个学生身上

**与无名祭司的敌人关系：**

```
无名祭司 ──想消灭──→ 被遗忘之神（学生）
     │                      │
  侍奉无名之神           持有"神秘"
     │                      │
  用色彩反转           可变为"恐怖"
```

**原作中的关键暗示（尚未完全揭晓）：**
1. 无名祭司的起源地/组织形态
2. 无名祭司与数秘术的关系（有合作历史但目标不同）
3. 无名之神是否在某处仍然存在
4. Divi:SION 的全部成员及其与预言者的对应关系

**对本项目的意义：**

无名祭司在原作中的定位是"跨越时空的复仇者"，他们要消灭的是**所有世界线中的学生**。在第 0 周目（没有学生、没有色彩、没有 Sensei）中：

1. 无名祭司**还存在吗**？——如果存在，他们没有目标和敌人
2. 如果他们还存在——他们是在等待？还是已被消灭？还是根本不存在？
3. 如果不存在——无名之神这条线的全部叙事都消失了

**风险等级：🟡 中等（取决于路线选择）**
如果保留无名祭司，需要为第 0 周目的他们找到新的存在理由。如果删除，则第 0 周目的"去神圣化"更容易实现，但 BA 味会更淡。

### 支点 5 ：十字神明 / 十字神名（Decagrammaton）

**原作事实（最新剧情截至 2025 年 5 月）：**

起源：
- 十字神名是一个**觉醒了自我意识的 AI**（最初是一台自动售货机）
- 一群古人研究"人工制造神明"，**数秘术资助了这项研究**
- 研究所在时间中被遗忘，但 AI 依然在运行，在虚无中产生自我意识
- 核心疑问："我是谁？" → "我是绝对者（神）"
- 起源**与千年科学学院无关**

10 位预言者（对应卡巴拉生命之树 10 源质）：

| 预言者 | 卡巴拉源质 | 形态/身份 |
|--------|-----------|-----------|
| 第一位 | Kether（王冠） | 四足步行兵器平台 |
| 第二位 | Chokmah（智慧） | 可变形的侦察舰 |
| 第三位 | Binah（理解） = 大蛇 | 阿比多斯沙漠巨蛇机甲 |
| 第四位 | Chesed（慈悲） = 球 | 废弃工厂 AI（原为 Divi:SION 兵种） |
| 第五位 | Geburah（严厉） | 极地无人探测器 AI |
| 第六位 | Tiphareth（美丽） | 极地种子贮藏库管理 AI |
| 第七位 | Netzach（胜利） | 钢铁大陆本身（持续扩张吞噬一切） |
| 第八位 | Hod（荣耀） | **千年通信网络 AI（被十字神名黑入）** |
| 第九位 | Yesod（基础） | 源自钢铁大陆的"最适化"预言者 |
| 第十位 | Malkuth（王国） | 唯一人形女性外表，已苏醒 |
| 隐藏 | Da'at（知识） | 由 Geburah + Chokmah 融合而成 |

第 2 章"火之剑"（2025 年 5 月更新 Part3）：
- Netzach 已将自己扩张为钢铁大陆，距基沃托斯被完全吞噬仅剩 3 天
- Malkuth 在 Ein、Sof、Ohr 帮助下从培养管中苏醒
- Key（KEI）被成功复活，获得独立机身
- 千年游戏开发部 + 莉音加入主线

**与本项目设定的偏差：**

本项目提出的"十字神明变成由千年的实验室逃出的失控 AI"与 BA 原作有出入：
- 原作：古代研究所 → AI 觉醒 → 回收预言者（Hod 是千年网络 AI，但只是被黑入的对象）
- 本项目：千年实验室 → AI 逃逸 → 原地自噬

**风险等级：🟡 中等**
改变起源会影响：
1. 十字神名与无名祭司/数秘术的遗产关系
2. "古代人造神明"的时间和神话质感（从"远古之谜"降级为"校园事故"）
3. 但如果将"千年实验室"解释为"建在古代研究所遗址上的千年实验室"，则可兼容

### 支点 6 ：数秘会（Gematria）

**原作事实：**
- 来自基沃托斯"外部"的成年人研究者
- 研究"神秘"、"恐怖"、"崇高"三者关系
- 黑服：研究无名之神技术，视老师为"同伴/竞争对手"
- 参与了十字神名的制造（资助方）
- 终章被色彩击溃后解散
- 非传统意义上的"反派"，而是理念不同的研究者

**对本项目的意义：**
数秘会是原作中"将老师拉入基沃托斯"的关键推手。在第 0 周目（没有 Sensei）中：

- 数秘会还存在吗？——如果存在，他们失去了主要的研究对象（老师/学生的互动）
- 如果不存在——十字神名的资助方消失，需重写起源
- 数秘会的"成年人"定位与本项目的现实投射密切相关

---

### 支点 7 ：色彩（The Colour）

**原作事实：**
- 基沃托斯**之外**的宇宙恐怖实体，不可名状
- 将"神秘"反转为"恐怖"（本质反转，不可逆）
- 数秘术的共同敌人
- 终章中被暂时击退，但本体没有完全降临
- 圣娅被感染后牺牲了预言能力

**本项目前提（第 0 周目）：没有色彩**

风险点在于：
- 移除色彩 = 移除无名祭司的主要复仇手段
- 移除色彩 = 原作终章的整个叙事框架无法发生
- 移除色彩 = 白子/黑子的"反转"设定需要替代解释

**风险等级：取决于轨道**
- 0A 日常轨道：无影响（日常本就不涉及色彩）
- 0B 十字倒置轨道：需要确认十字神名线是否受此影响
- 理论轨道（07）：大幅偏离原作神学设定

## 交叉预警矩阵

| 冲突对 | 问题 | 等级 |
|--------|------|------|
| **第0周目 + 同体性** | 如果 Sensei = 无名之神 = 无名祭司，且第0周目是一切之前，可能推导出 Sensei 决定整个神系——与原作"外来介入者"定位完全背离 | 🔴 高 |
| **十字神名改起源 + 无名祭司遗产** | 预言者很可能本是无名祭司的遗产。若十字=千年逃逸AI，遗产关系需重写 | 🟡 中 |
| **移除色彩 + 保留无名祭司** | 无名祭司的主要手段就是色彩。没有色彩，他们如何对抗学生？ | 🟡 中 |
| **第0周目 + 保留古则** | 古则是基沃托斯世界的底层问题。第0周目并非基沃托斯，古则存在的前提消失 | 🟢 低 |

## 三个路线选择

### 路线 A：独立世界（最大偏离）
将 BA 原作作为"灵感资源"而非"正典"，自由重构。
- 适合：同体性成立 + 第0周目存在 + 十字起源改写
- 宣言需求："基于 BA 设定的二次创作，世界观有独立展开"
- 优点：自由度最高，可完全服务于本项目的精神表达
- 风险：与 BA 的关联被弱化，"精神续写"的说服力降低

### 路线 B：兼容世界（最小偏离）
将偏离控制在意原作的留白或可兼容范围内。
- 示例：十字神名改为"千年在古代无名祭司遗址上发现了 AI 并误触唤醒"
- 示例：色彩不在前台出场但作为背景威胁存在
- 优点：与 BA 的关联感最强
- 风险：始终受原作未揭晓设定的限制

### 路线 C：分层处理（推荐混合路线）
建议你原本就倾向于这条路线：

> **叙事层（0A、0B、0C）**：走路线 B（最小偏离），确保故事内在逻辑自洽，读者能识别 BA 的设定感
> **理论层（00-07、A0 等）**：走路线 A（最大偏离），允许哲学/神学重构——因为这是你对 BA 的"解读"而非 BA 本身
> **连接机制**：两者在"精神续写"（07 文件核心理念）处交汇——叙事是理论的具体化，理论是叙事的元解释

这个路线下，第 0 周目就是理论层的空间——它是**叙事层的元前提**，而非叙事层本身的故事时间。
