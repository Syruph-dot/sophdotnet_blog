# Pixel Sort Glitch Studio

Glitch art tool — drag image, pixel sort + RGB shift.

## MVP

- Drop/click image upload
- Horizontal/vertical/bidirectional pixel sorting by brightness threshold
- RGB channel shift (chromatic aberration)
- Sort angle skew
- 5 presets (Glitch 1-3, Vaporwave, Ghost)
- Randomize + Download PNG
- Single-file HTML, zero deps

## Usage

Open `index.html` in any modern browser. Drop image → tweak sliders → download glitch.
