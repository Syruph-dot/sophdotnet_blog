# 🎲 粒子引力游乐场 (Particle Gravity Playground)

交互式 Canvas 物理模拟器 — 点击生成粒子，观察它们在引力、碰撞和网络连线下的舞蹈。

## 功能

- 🖱️ **点击生成** — 在画布任意位置点击，随机生成彩色粒子
- 🌍 **引力模拟** — 所有粒子受重力影响向下坠落（G 键切换）
- 💥 **粒子碰撞** — 粒子之间相互碰撞弹开（C 键切换）
- ✨ **轨迹模式** — 粒子运动留下绚丽拖尾（T 键切换）
- 🎆 **烟花模式** — 一键绽放满屏烟花（F 键触发）
- 🌀 **吸引子** — Shift+点击放置重力吸引子，粒子被吸引靠近；再次 Shift+点击移除（A 键切换模式）
- 🔗 **粒子连线** — 近距离粒子自动绘制渐变发光连线，形成神经网络般视觉效果（N 键切换）
- 🖐️ **拖拽交互** — 用鼠标拖拽粒子改变运动轨迹
- 🎨 **随机颜色** — 每个粒子随机 HSL 颜色
- 🧹 **一键清空** — 空格键清除所有粒子和吸引子

## 技术栈

- 纯 HTML/CSS/JS，单文件应用
- HTML5 Canvas 2D 渲染
- 自实现物理引擎（引力 + 弹性碰撞 + 吸引子力场）

## 玩法

1. 打开 `src/index.html`
2. 在画布上疯狂点击生成粒子
3. 按 **N** 开启连线模式，欣赏粒子网络
4. 按 **A** 进入吸引子模式，Shift+点击放置力场中心
5. 拖拽粒子玩玩
6. 按 **F** 看一场烟花表演
7. 观察它们碰撞、弹跳、连线、被吸引子拉扯
8. 按空格键清空重新开始
