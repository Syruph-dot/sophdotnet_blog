# ASCII Webcam 🎥➡️🔤

实时将摄像头画面转换为 ASCII 字符画的网页工具。

## 功能

- 📷 调用摄像头，实时捕获画面
- 🎨 将每个像素块映射为 ASCII 字符（亮度 → 字符密度）
- 🎚️ 可调节：分辨率（字符密度）、对比度、字符集
- 🖤 支持暗色/亮色主题切换
- 📸 点击截图保存当前 ASCII 画面为 PNG
- ⏸️ 暂停/播放控制

## 技术栈

- 纯 HTML/CSS/JS，单文件
- WebRTC (getUserMedia) 获取摄像头
- Canvas 进行像素处理
- 零依赖，浏览器直接打开即可

## 使用方法

直接用浏览器打开 `index.html`，授权摄像头即可开始。
